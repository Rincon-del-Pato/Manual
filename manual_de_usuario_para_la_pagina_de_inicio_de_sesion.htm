1. Manual de Usuario para la Página de Inicio de Sesión

 
Este manual proporciona instrucciones detalladas sobre cómo utilizar la página de inicio de sesión del sistema "Rincón del Pato".
El objetivo principal es guiar al usuario para que pueda acceder al sistema o registrarse como nuevo usuario.
 
1. Componentes de la Interfaz
       La página de inicio de sesión consta de los siguientes elementos:
1. Logo y Nombre del Sistema:
  • En la parte superior, se encuentra el logotipo del sistema con el texto "Rincón del Pato".
    2. Campos de Inicio de Sesión:
  • Email: Campo para ingresar la dirección de correo electrónico.
  • Password: Campo para ingresar la contraseña asociada al usuario.
    3. Opciones de Inicio de Sesión:
  • Recordar contraseña: Una casilla que puede marcar para que el sistema recuerde sus credenciales en futuras sesiones.
    4. Botón "Ingresar":
  • Al presionarlo, se envían los datos ingresados para acceder al sistema.
    5. Enlace para Registrar Nuevo Usuario:
  • Una opción para los usuarios que aún no tienen una cuenta y desean crear una.
     
 
2. Cómo Iniciar Sesión
      Siga los pasos a continuación para acceder al sistema:
1. En el campo Email, escriba su dirección de correo electrónico registrada.
2. En el campo Password, escriba su contraseña.
3. Si desea que el sistema recuerde sus credenciales, marque la casilla Recordar contraseña.
4. Haga clic en el botón Ingresar.
  • Si las credenciales son correctas, será redirigido al panel principal del sistema.
  • Si las credenciales son incorrectas, aparecerá un mensaje de error.
     
3. Cómo Registrar un Nuevo Usuario
      Si es la primera vez que accede al sistema:
1. Haga clic en el enlace Registrar nuevo usuario.
2. Complete el formulario de registro con sus datos personales, como nombre, email y contraseña.
3. Haga clic en el botón para finalizar el registro.
4. Una vez registrado, utilice las credenciales creadas para iniciar sesión.
 
4. Recomendaciones de Seguridad
  • Asegúrese de usar una contraseña segura (al menos 8 caracteres, combinando letras, números y símbolos).
  • Nunca comparta sus credenciales con terceros.
  • Cierre sesión si utiliza un dispositivo compartido.
