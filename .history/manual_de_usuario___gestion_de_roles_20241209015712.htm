2. Manual de Usuario - Gestión de roles

 
El módulo "Roles" permite administrar los distintos perfiles de usuario del sistema, cada uno con niveles específicos de acceso y funcionalidades. Este manual está diseñado para guiar al Super Admin y otros usuarios autorizados en la gestión de roles del sistema.
 
1. Descripción de la Interfaz
Elementos principales:
1. Barra de búsqueda:
  • Campo para buscar roles existentes por su nombre.
    2. Filtros por fecha:
  • Dos campos de selección de fecha ("Desde" y "Hasta") que permiten filtrar roles creados dentro de un rango específico.
    3. Tabla de roles:
  • N°: Número de identificación del rol.
  • Rol: Nombre del rol (e.g., Gerente General, Mesero, Cocinero).
  • Acciones: Opciones disponibles para cada rol:
  • Editar: Modificar los detalles del rol.
  • Eliminar: Eliminar el rol de manera permanente.
    4. Botón "Exportar":
  • Permite descargar la lista de roles en diferentes formatos (e.g., Excel, PDF).
    5. Botón "Agregar Roles":
  • Redirige a un formulario para crear un nuevo rol.
     
     
2. Funciones por Rol
Super Admin:
  • Acceso total al sistema, incluidas las configuraciones críticas.
  • Funciones principales:
  • Crear, editar y eliminar roles.
  • Configuración de permisos y acceso del sistema.
  • Gestión de todos los módulos.
Administrador:
  • Gestión general del sistema, con acceso restringido a configuraciones críticas.
  • Funciones principales:
  • Administración de usuarios y roles (sin acceso a permisos críticos del Super Admin).
  • Supervisión de operaciones generales.
Gerente General:
  • Responsable de la gestión de operaciones de alto nivel.
  • Funciones principales:
  • Acceso a informes de desempeño.
  • Supervisión de roles como Mesero, Cocinero y Cajero.
Mesero:
  • Función orientada a la gestión de pedidos y mesas.
  • Funciones principales:
  • Tomar pedidos de los clientes.
  • Asignar pedidos a mesas específicas.
  • Consultar el estado de los pedidos.
Cocinero:
  • Función específica para la vista de cocina y preparación de pedidos.
  • Funciones principales:
  • Ver pedidos enviados desde los meseros.
  • Actualizar el estado de preparación de los platillos.
3. Instrucciones para la Gestión de Roles
3.1. Crear un nuevo rol
1. Haz clic en el botón "Agregar Roles".
2. Completa el formulario con:
  • Nombre del rol.
  • Descripción de las funciones asociadas.
  • Permisos asignados (según las políticas del sistema).
    3. Guarda los cambios haciendo clic en el botón "Guardar".
3.2. Editar un rol existente
1. Localiza el rol que deseas modificar en la tabla.
2. Haz clic en el ícono ✏️ Editar.
3. Realiza los cambios necesarios en el formulario.
4. Guarda los cambios.
3.3. Eliminar un rol
1. Localiza el rol en la tabla.
2. Haz clic en el ícono 🗑️ Eliminar.
3. Confirma la acción cuando el sistema lo solicite.
  • Nota: La eliminación es irreversible.
3.4. Exportar la lista de roles
1. Haz clic en el botón "Exportar".
2. Selecciona el formato deseado (e.g., Excel, PDF).
3. Descarga el archivo generado.
4. Recomendaciones Generales
  • Asegúrate de asignar roles solo a usuarios autorizados.
  • Realiza copias de seguridad antes de eliminar roles importantes.
  • Capacita a los usuarios en las funciones específicas de su rol para evitar errores operativos.
