8. Manual de Usuario - Crear Producto

 
El módulo Crear Producto permite registrar nuevos productos en el sistema. Esto es útil para mantener un inventario organizado y facilitar la gestión del menú, adaptándose a las necesidades de tu negocio.
 
1. Descripción de la Interfaz
Formulario "Crear Producto"
  • Nombre del producto: Campo de texto donde se ingresa el nombre completo del producto (por ejemplo: "Pizza Margherita").
  • Categoría: Menú desplegable para seleccionar la categoría a la que pertenece el producto (por ejemplo: "Pizzas", "Bebidas").
  • Descripción: Campo de texto para detallar las características del producto (por ejemplo: "Pizza con base de queso, tomate y albahaca fresca").
  • Tipo de producto: Opción para elegir entre:
  • Para elaborar: El producto necesita preparación antes de ser vendido.
  • Preenvasado: El producto ya está listo para su venta.
    5. Tamaños y precios:
  • Seleccionar tamaño: Menú desplegable para elegir el tamaño del producto (por ejemplo: "Pequeño", "Mediano", "Grande").
  • Precio: Campo de texto donde se especifica el precio correspondiente a cada tamaño.
  • Agregar tamaño: Botón que permite añadir más opciones de tamaño y precio.
  • Eliminar tamaño: Ícono de papelera que elimina un tamaño y su precio correspondiente.
    6. Disponible para venta: Interruptor para indicar si el producto estará disponible para la venta inmediatamente después de ser registrado.
    7. Botones:
  • Guardar: Registra el nuevo producto en el sistema.
  • Cancelar: Limpia los campos del formulario sin guardar la información.
     
     
     
2. Instrucciones de Uso
2.1.  Crear un nuevo producto
  • Accede al módulo Productos desde el menú principal.
  • Selecciona la opción Crear Producto.
  • En el formulario que aparece:
  • Completa el campo Nombre del producto (por ejemplo: "Pizza de Pepperoni").
  • Selecciona la Categoría correspondiente (por ejemplo: "Pizzas").
  • Ingresa una Descripción que resuma las características del producto.
  • Selecciona el Tipo de producto (por ejemplo: "Para elaborar").
  • En la sección Tamaños y precios:
  • Escoge un tamaño en el menú desplegable.
  • Ingresa el precio correspondiente.
  • Si necesitas añadir más tamaños, haz clic en Agregar tamaño.
  • Para eliminar un tamaño, utiliza el ícono de la papelera.
Activa o desactiva el interruptor Disponible para venta según corresponda.
4. Verifica que toda la información esté correcta.
5. Haz clic en el botón Guardar para registrar el producto en el sistema.
 
2.2. Cancelar la creación de un producto
  • Si decides no guardar la información, haz clic en el botón Cancelar.
  • Esto limpiará los campos del formulario sin guardar los datos ingresados.
     
3. Recomendaciones Generales
  • Nombre descriptivo: Utiliza nombres claros y fáciles de identificar para tus productos. Ejemplo: En lugar de "Pizza especial", usa "Pizza especial de jamón y champiñones".
  • Categoría adecuada: Asegúrate de que el producto esté correctamente clasificado para facilitar su búsqueda.
  • Precios correctos: Revisa que los precios sean acordes al tamaño y categoría del producto.
  • Evitar duplicados: Antes de crear un producto, verifica que no exista ya uno con características similares en el sistema.
4. Solución de Problemas
  • Error al guardar: Si aparece un mensaje de error, revisa que:
  • Todos los campos obligatorios estén completos.
  • No existan productos con el mismo nombre.
  • Formulario no responde: Recarga la página e intenta nuevamente agregar al proveedor.
  • Recarga la página e intenta nuevamente.
  • Si el problema persiste, contacta al administrador del sistema.
