4. Manual de Usuario: Crear categoría

 
El módulo Agregar Categoría permite registrar y organizar las categorías dentro del sistema. Cada categoría debe incluir un nombre identificativo y una descripción que detalla su propósito. Este manual está diseñado para guiar al usuario en la creación y manejo de categorías de manera eficiente.
 
1. Descripción de la Interfaz
Formulario "Agregar Categoria"
  • Nombre Categoría: Campo de texto donde se ingresa el nombre de la categoría. Este nombre debe ser único y representativo (por ejemplo: "Bebidas", "Comidas Rápidas").
  • Descripción: Campo de texto donde se detalla la función o propósito de la categoría.
  • Botones:
  • Crear: Registra la categoría en el sistema.
  • Cancelar: Limpia los campos del formulario sin realizar cambios
 
 
2. Instrucciones de Uso
2.1. Agregar una nueva categoría
1. Accede al módulo de Categorías.
2. Localiza y selecciona la opción "Agregar Categoría".
3. En el formulario que aparece:
  • Completa el campo Nombre Categoría (por ejemplo: "Postres").
  • Añade una Descripción breve pero clara (por ejemplo: "Categoría para productos dulces como tortas, helados y pasteles").
    4. Haz clic en el botón Guardar para registrar la nueva categoría.
     
2.2. Cancelar la creación de una categoría
1. Si decides no guardar la categoría, haz clic en el botón Cancelar.
2. Esto limpiará los campos del formulario y no se guardarán los datos ingresados.
 
3. Recomendaciones Generales
1. Nombre único: Asegúrate de que el nombre de la categoría no se repita, ya que esto puede generar errores en el sistema.
2. Descripción clara: Proporciona una descripción precisa para facilitar la comprensión del propósito de la categoría.
3. Revisión previa: Antes de guardar, verifica que los datos ingresados sean correctos..
 
4. Solución de Problemas
  • Error al guardar: Si aparece un mensaje de error, revisa que:
  • El campo Nombre Categoría no esté vacío.
  • No exista otra categoría con el mismo nombre.
  • Formulario no responde: Recarga la página y vuelve a intentar agregar la categoría.
